# MVC Project Template

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run:
   ```bash
   npm start
   ```
3. Open browser: [http://localhost:3000](http://localhost:3000)

## Folder Structure
- models/ → Data logic
- controllers/ → Request handling
- views/ → UI templates
- routes/ → URL mappings
- public/ → CSS, JS, Images
- screenshots/ → Evidence for submission
